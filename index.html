API Endpoints
FastAPI 기반 파이프라인 제어 API
POST
/api/pipeline/trigger
파이프라인 시작. topic + channel_id 전송. workflow_id 반환.
GET
/api/pipeline/status/{id}
실시간 진행 상태. 현재 step, scene 진행률, 비용, YouTube URL 포함.
GET
/api/pipeline/cost/{id}
비용 상세. fal.ai 등 서비스별 breakdown + 총액.
DELETE
/api/pipeline/{id}
파이프라인 취소. Temporal workflow 종료.
